1.1、操作步骤：

1.根据场景下拉框，选择自己的明文模拟数据；

2.点击生成测试按钮，将左边的明文进行加密加签操作；

3.点击收单按钮，如果能正常跳转即完成收单模拟。


1.2、收单特殊字段详解
字段名 变量名 类型 说明 可空
merchantOrderId 商家订单号 String 商家自己的订单号，不可重复 N
expiredDate 订单失效时间 String 格式化为yyyyMMddHHmmss，这个时间在收单时，根据商家需要，可以自行设置订单的失效时间（例如：失效时间可以设置为订单时间后一天） N
merchantUserId 用户唯一标识 String 一定要保证商家每个用户该字段的唯一性（重中之重） N
